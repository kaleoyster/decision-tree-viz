####  Clone
```zsh
git clone https://github.com/kaleoyster/decision-tree-viz.git
```

#### Run requirements.txt

```zsh
pip install -r requirements.txt
```

#### Run the decision tree model

```zsh
python3 src/decision_tree.py
```

#### Run http server 

```zsh
python -m http.server
```

#### View visualization

```zsh

```
